
# Obsidian-Signed-Remarks
Signed Remarks Plugin for Obsidian

this plugin  allows you to add custom signed remarks, annotations, and visual markers to your notes with personalized images and colors.

### üöÄ Installation
From Obsidian Community Plugins

- **Disable Restricted Mode**: Open Obsidian **Settings** > **Community plugins**, and turn off **Restricted Mode**.
- **Download the Release**: Go to the GitHub repository of the plugin and navigate to the **Releases** section. Download the latest [obsidian-signed-remarks.zip](https://github.com/rollenso/Obsidian-Signed-Remarks/releases/tag/v1.0.0) file.
- **Locate the Plugins Folder**:
    - Open your Obsidian vault in your computer's file explorer.
    - Navigate to the hidden folder: `<your-vault-name>/.obsidian/plugins/`.
- **Extract the Files**:
    - Unpack the downloaded zip file into the `plugins` folder.
* Reload Obsidian and enable plugin in **Obsidian Settings**

Manual Installation

### üìñ Usage Guide
Basic Usage

Open the settings:

        Go to Settings ‚Üí Community plugins

        Find Signed Remarks and click the gear icon

    Create your first sign:

        Click "+"

        Enter an image URL or base64

        Set a border color (hex code)

    Insert a sign in your note:

        Open command palette (Ctrl/Cmd + P)

        Search for "Insert Sign #1"

        Press Enter to insert the sign
